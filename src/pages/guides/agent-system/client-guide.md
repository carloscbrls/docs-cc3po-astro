# Client Guide

Everything you need to know to work with your AI agent team.

## Getting Started

### Your First Week

**Day 1: Access**
- You'll receive login credentials
- Access your dashboard at your custom URL
- Explore the agents panel

**Day 2-3: Training**
- We'll walk through your business
- You'll share your preferences
- Agents will learn your style

**Day 4-5: First Tasks**
- Agents will start small tasks
- You'll review and give feedback
- Agents will adjust

**Day 7: Full Operation**
- Agents work 24/7
- You check dashboard weekly
- You provide input as needed

## How to Request Tasks

### Via Dashboard
1. Log in to your dashboard
2. Go to "New Task"
3. Describe what you need
4. Select which agent(s)
5. Submit

### Best Practices

**Be Specific:**
❌ "Do marketing"
✅ "Create 5 LinkedIn posts for my HVAC business about spring maintenance, focusing on energy efficiency"

**Provide Context:**
- Who is your audience?
- What's your goal?
- Any constraints?

**Set Deadlines:**
- When do you need it?
- Is it urgent or routine?

## Understanding Your Dashboard

### Status Indicators
| Status | Meaning |
|--------|---------|
| 🟢 Running | Agent is working |
| 🟡 Waiting | Needs your input |
| 🔴 Blocked | Issue detected |
| ✅ Complete | Task finished |

### Activity Feed
- Shows all agent actions
- Timestamps for each task
- Results and outputs

### Metrics
- Tasks completed
- Content created
- Leads found
- Engagement rates

## Giving Feedback

### What Helps
- "This is great, keep going"
- "Needs more [X]"
- "Try [Y] instead"
- "Make it more [tone]"

### What Doesn't Help
- "Do better"
- "Fix it"
- "I don't like it"
- Vague criticism

### How to Give Feedback
1. Go to completed task
2. Click "Give Feedback"
3. Be specific
4. Agent learns from it

## Common Questions

### How often should I check in?
- Weekly: Recommended
- Daily: If actively launching
- As needed: For urgent tasks

### Can agents make decisions?
- Routine decisions: Yes
- Major decisions: They'll ask you
- Financial decisions: Always ask

### What if something goes wrong?
1. Check dashboard for alerts
2. Read the error message
3. Contact support if needed

### Can I pause agents?
Yes. Go to Settings > Agent Controls > Pause

### How do I add new tasks?
Dashboard > New Task > Describe > Submit

## Maximizing Value

### 1. Batch Similar Tasks
Instead of daily requests, batch them:
- Weekly content requests
- Monthly lead reviews
- Quarterly audits

### 2. Provide Templates
Share your best:
- Email templates
- Social post formats
- Brand guidelines
- Voice/tone examples

### 3. Review Regularly
Set a weekly time to:
- Review completed work
- Give feedback
- Adjust priorities

### 4. Trust the Process
- Agents learn over time
- First attempts may need adjustment
- Patience pays off
