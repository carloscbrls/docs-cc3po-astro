## Business Decisions

### Central Valley Locals (CVL)
- **What:** Local business directory for Central Valley
- **Why:** Proof of concept, community foundation
- **Revenue:** Listings ($29-$399/mo), leads, ads
- **Status:** Building

### CC3PO Agent System
- **What:** Sell agent systems to others
- **Why:** Scale beyond one person
- **Pricing:** $500-2500 setup + $99-499/mo
- **Status:** Launching

### Client Work
- **What:** IT consulting, websites, automation
- **Why:** Immediate revenue
- **Priority:** When it pays well

## Technical Decisions

### Hosting
- **Netlify** for static sites (CVL, offers)
- **SiteGround** for WordPress (cc3po.com)
- **DigitalOcean** for client servers

### Database
- **Supabase** for CVL (PostgreSQL)
- **JSON files** for simple data

### Automation
- **Zernio** for social media (all platforms)
- **JotForm** for forms
- **Twilio** for SMS/voice
- **OpenClaw** for agent orchestration

### Content
- **WordPress + Elementor** for cc3po.com
- **Standalone HTML/PHP** for subdomains
- **Netlify** for deployment

## What Works

### For Content
- Educational posts perform best
- Stories and case studies engage
- Clear CTAs convert
- Consistent posting builds trust

### For Lead Gen
- Local focus wins
- Score before outreach
- Follow up within 24 hours
- Multi-touch (SMS + email + voice)

### For Clients
- Clear expectations upfront
- Regular check-ins
- Document everything
- Over-communicate

## What Doesn't Work

### For Content
- Overly promotional
- Inconsistent posting
- Ignoring engagement
- Generic content

### For Lead Gen
- Cold calling without research
- Single-touch outreach
- No follow-up
- Wrong timing

### For Clients
- Vague scope
- No regular communication
- Assuming understanding
- Skipping documentation

## Lessons Learned

### Technical
- **Cache issues:** SiteGround nginx cache breaks updates — use standalone HTML
- **Model deprecation:** APIs change — use aliases
- **API limits:** Free tiers run out — monitor usage

### Business
- **Pricing:** Start higher — easy to discount, hard to raise
- **Clients:** Not all clients fit — fire bad ones
- **Scope:** Define clearly — scope creep kills margins

### Process
- **Automation first:** If you do it twice, automate it
- **Document everything:** Future you needs help
- **Test before deploy:** Always
