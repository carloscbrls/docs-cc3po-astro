## Quality Control

### Content Review Checklist
- [ ] Spelling/grammar checked
- [ ] Links working
- [ ] Images optimized
- [ ] SEO meta complete
- [ ] CTA included
- [ ] Brand voice consistent
- [ ] Mobile responsive
- [ ] Load time acceptable

### Lead Quality Checklist
- [ ] Contact verified
- [ ] Business validated
- [ ] Priority scored
- [ ] Source documented
- [ ] Notes added
- [ ] Added to CRM

## Troubleshooting

### Common Issues

| Issue | Solution |
|-------|----------|
| Agent not responding | Check API keys, restart session |
| Content quality low | Refine prompt, add examples |
| Leads not converting | Review scoring criteria |
| Social posts failing | Check platform connections |
| Emails bouncing | Verify email list, check DNS |
